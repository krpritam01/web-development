1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    Styling and layout of web pages
    • CSS enables the creation of responsive web designs
    • CSS is primarily used to style and present the visual aspects of a web page.
    • CSS provides powerful layout techniques.

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    Adding interactivity and functionality to web pages
    • JavaScript is primarily used for adding interactivity and behavior to web pages.
    • JavaScript provides event-driven programming capabilities.
    • JavaScript enables the manipulation and processing of data on the client side.
    • JavaScript facilitates integration with various
      APIs (Application Programming Interfaces) to access external services and retrieve data.
